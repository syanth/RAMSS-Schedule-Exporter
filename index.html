RAMSS Schedule Exporter

This is an unoffical tool that helps Ryerson University students easily export their class schedule into the iCalendar (.ics) format compatible with Apple Calendar, Google Calendar, Outlook, and most other calendar applications.

This tool was created by Syanth Paramananthan. You can view the source code on Github.

Note: The calendar will have classes scheduled on the day before they actually start. Manually delete those.

How to:

  • 1. Log onto my.ryerson.ca and open RAMSS
  • 2. Go to Self Service --> Enrollment --> My Class Schedule
  • 3. Select which semester you want to export
  • 4. Select the content starting from the first class to the last class
  • 5. Copy paste your schedule here:
  • 6. Click "Download Calendar"
